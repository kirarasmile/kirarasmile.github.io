实装了魔幻评论系统(附教程)

2019-08-28 博客维护
Word count: 1.3k | Reading time: 4min

博客已经实装了评论系统….得益于这个主题的作者留好的模板，我很轻松的就搞定了，模板里一共有四个选择：Livere、Disqus、倡言、Gitment，目前博客里使用的是Livere（即来必力），它能正常的运行（要挂梯子）并且有多种登录渠道。


事实上昨天晚上我还尝试了其他几种评论系统，下图是Disqus的，同样要挂梯子才可以看到，与Livere功能是相同的，但是个人觉得Disqus界面更好看一点（我懒得改回去了），虽然登录渠道比较少。

Gitment原本是我的最优选择，但是他挂了，我把ID和秘钥填入之后虽然能正确显示评论框架，但却无法正常的登录与发表评论，在网上查阅之后得出结论是我连上的评论服务器是Gitment作者的服务器，但是这个服务器挂了，而目前我对这整个博客框架几乎毫无修改头绪，因此我无从得知到底要修改何处才能将Gitment的评论挂靠在我的git仓库里头，这个问题可能要放在后期研究了。

倡言是我发现Gitment挂了之后的第二选择，但是他居然要我给他个备案号……我的这git page搭的blog上哪给他整去（网上貌似有教程）…..

HEXO搭建评论系统教程

Livere

打开Livere的官网，https://www.livere.com/打不开的话需要挂上梯子，打开之后你会发现网页里头是友好的中文
…但是不要被骗了。接下来你在这个网站的所有操作依然需要挂好梯子，第一步是要注册账号，（我在注册的时候发生了一件很迷惑的事情，明明我输入的验证码是正确的，但是他一直给我弹验证码错误，我重复了注册步骤好几次都是这样，于是我扭头去尝试Disqus，待我回来准备操作的时候却发现我能直接登录了….真是魔法）

当你注册完成登录之后，点击主页上方导航栏的安装按钮（install，如果你选的是英文的话，页面最下方有语言切换选项），选择免费版（city版），点击现在安装，

然后你就会转入这个信息填写页面，

  1. 第一行填写你的博客的完整地址
  2. 第二行填写你的博客名称
  3. 第三行选择个人网站
  4. 点击申请获取代码

然后你就会获得一串代码，如图：

打码部分便是我们需要的重点，如果你使用的博客模板为比较主流或是作者本身有预留评论功能的话，打开你的themes文件夹下对应正在使用的模板文件夹中的_config.yml,在里面找到# comment下方的livere_uid，在后面填入刚才你在代码页面我打码部分的代码（记得空一格）,保存之后部署到blog，就大功告成了

如果你的博客没有预留评论功能的话，你就需要到搜索引擎上查找如何将评论功能模块加入你的blog了。（主要原因是我发现我找遍我现在的博客主题里头的所有文件夹，也没找到网上各种教程的对应文件，应该是原作者打包在某个地方了。）

Disqus

同样，打开Disqus官网https://disqus.com/，在注册登录之后点击主页的get start按钮，然后你会来到这个页面：

点击I want to install Disqus on my site，然后你就会进入这个页面
需要填写的有三个框，

  1. Website Name，这个是你一会要填入你_config.yml文件里的short name
  2. Category是你的博客种类，这个随你选
  3. Language就不用说了。
  4. 点击Creat Site

跳转到这个页面，往下拉
点击Subscribe Now，跳转到下一个页面

因为我们使用的是GIT PAGE，不属于上面那些个的任何一个，所以我们点击最下面我用红色箭头标出来的按钮，跳转到下一个页面

在这个页面往下拉，页面的右下角有一个Configure，点击再进入下一个页面

如果你的Website Name没有重复，在这个页面里，我们只需要填写一个信息框就行了：

  1. Website URL，填写你的blog地址
  2. 如果你的Website Name重复了，那么你接下来要填入配置文件的shortname会变成这个页面Website Name中所显示的文字

ps：记住的你的shortname

接下来操作和上面Livere结束部分一样，打开你的themes文件夹下对应正在使用的模板文件夹中的_config.yml,在里面找到# comment下方的disqus_shortname，在后面填入刚才你在代码页面我打码部分的代码（记得空一格）,保存之后部署到blog。
